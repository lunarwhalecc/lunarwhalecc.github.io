Connecting to IRC

  • Server: irc.irchighway.net
  • Channel: #ebooks
  • Searching: In channel, type @search search term
  • Unzip and open the text file with search results, find the book you want
  • Copy and paste the line into the channel for the book you want (I recommend always getting epub, it's the most standard/portable format), eg
    !Dumbledore Adrian Tchaikovsky - [Children of Time 01] - Children of Time (epub).epub
  • Unzip/unrar the book if necessary

Add to Calibre

  • Add the book to Calibre
  • Can update the metadata if needed/wanted
  • Connect Kindle and send to device, converting as needed
  • Eject Kindle and enjoy the book
